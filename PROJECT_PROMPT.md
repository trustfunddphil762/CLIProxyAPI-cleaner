# 项目提示词 / Project Prompt

如果你要把这个项目继续交给 AI 或协作者维护，可以直接贴下面这段：

```text
这是一个给 cpa_cleaner.py 使用的轻量 Web 控制台项目。

目标：
1. 保持“轻量、可直接部署、尽量少依赖”的风格；
2. 不要把线上域名、管理密钥、账号 ID、真实密码、Cookie 等敏感信息写入仓库；
3. 所有配置应通过 web_config.json 读取，页面保存后重启对应服务即可生效；
4. UI 优先移动端可读性，风格保持简洁、接近苹果风；
5. 报告展示优先摘要信息，不直接把大段原始 JSON 堆给用户；
6. 改动时优先兼容已有部署路径：/opt/cpa-cleaner 和 /cpa-cleaner/；
7. 如果新增功能，优先考虑：密码修改、多用户权限、报告筛选、告警通知、Docker 化。

技术约束：
- 后端保持 Python 标准库优先，不强依赖 Flask / FastAPI；
- 前端保持原生 HTML/CSS/JS，避免引入大型框架；
- systemd 服务文件应与页面保存的配置保持一致，不要再出现“页面改了但实际服务没生效”的问题；
- 任何展示给用户的命令预览都要注意隐藏敏感参数。

请在不破坏现有部署方式的前提下继续改进这个项目。
```
